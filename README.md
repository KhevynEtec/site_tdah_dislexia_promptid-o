# site_tdah_dislexia_promptidao

# Descrição:

Site focado em ajudar pessoas com TDAH e dislexia a estudar com mais facilidade. Possui um chatbot inteligente que responde dúvidas acadêmicas de forma simples, clara e objetiva, além de sugerir conteúdos educativos.

# Tecnologias:

HTML
CSS
JavaScript
n8n (integração com IA via webhook)

# Chatbot:

O chatbot é integrado ao n8n, que recebe as mensagens do usuário, processa com IA e retorna respostas adaptadas para facilitar o aprendizado.

# Como clonar o projeto:
git clone https://github.com/seu-usuario/seu-repositorio.git
cd seu-repositorio

Depois disso, basta abrir o arquivo index.html no navegador.

# Configuração (n8n)

Para o chatbot funcionar corretamente:

Configure um webhook no n8n
Substitua a URL do webhook no código JavaScript
Ative o workflow no n8n

# Objetivo

Tornar o aprendizado mais acessível, simples e inclusivo para pessoas com dificuldades de atenção e leitura.
