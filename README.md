# 📚 FocusIA - TDAH e Dislexia

Site focado em ajudar pessoas com TDAH e dislexia a estudar com mais facilidade.

Possui um chatbot inteligente que responde dúvidas acadêmicas de forma simples, clara e objetiva, além de sugerir conteúdos educativos.

---

## 🎯 Objetivo

Facilitar o aprendizado de pessoas com dificuldades de atenção e leitura, tornando o estudo mais acessível, leve e eficiente.

---

## 🛠️ Tecnologias

- HTML

- CSS

- JavaScript

- n8n (integração com IA via webhook)

---

## 🤖 Chatbot

O chatbot funciona com integração ao n8n.

Fluxo:
Usuário → Site → Webhook (n8n) → IA → Resposta → Site

Ele responde perguntas acadêmicas de forma simples e adaptada para TDAH e dislexia.
